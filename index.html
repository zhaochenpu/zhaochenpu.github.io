使用hexo与github搭建静态博客

前言：本文是一篇简单的独立博客搭建教程，内容不够完善，在我使用博客的过程中会不断实践并进行补充。

1.安装git

百度上有很多安装git的教程和git的下载链接，安装过程很简单，但有一个地方需要注意下：这个地方最好选中图中选项。以后操作会方便些。

2.安装node.js

下载node.js（node.js中文站偶尔抽风打不开，可以访问https://nodejs.org/en/）安装过程很简单。

3.安装hexo

找个地方新建个文件夹，在新建的文件夹内空白处右键，打开Git Bash

在命令行界面中执行如下命令安装hexo：

npm install -g hexo

然后输入如下命令自动生成建立网站所需要的文件：

hexo init

然后按照提示，运行如下命令安装node_modules：

npm install

这时hexo基本的安装就已经完成，运行如下命令启动本地hexo服务：

hexo server

然后在浏览器中打开http://localhost:4000/进行预览，这时可以看到Hexo已自动生成了一篇blog，停止服务按Ctrl+C。

4.新建文章

在上步git bash窗口中执行

hexo new "新建文章标题"

会在\source_posts目录下心生成一个.md为后缀的markdown文件。当然也可以直接在\source_posts中新建一个md文件。用markdown编辑器将其打开就可以写文章了。

markdown编辑器我使用的是markdownpad。

5.更换主题

hexo有很多主题可以从这里挑个。

以hexo-theme-next主题为例，在git bash窗口中执行如下命令将主题文件克隆到themes文件夹中：

git clone https://github.com/iissnan/hexo-theme-next themes/next

然后打开站点配置文件_config.yml，找到theme 字段，并将其值更改为 next。然后执行

hexo server

并在浏览器中打开http://localhost:4000/进行预览，看是否更换主题成功。
